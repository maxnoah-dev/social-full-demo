# Frontend - Social Demo

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env` and update API URL if needed:
```bash
cp .env.example .env
```

Default API URL is `http://167.71.200.72:3002`. Update it if your backend runs on a different port or URL.

3. Run development server:
```bash
npm start
```

The app will open at http://167.71.200.72:3003
